Description of program:

I developed a Java program for a jewellery store management system, leveraging custom-built internal data structures. The system enables seamless management of a jewellery store, comprising of multiple display cases, trays, and numerous jewellery items such as rings, watches, and necklaces. Each jewellery item consists of various materials/components, including gold, platinum, diamonds, and more, with varying quantities and qualities measured in grams or carats. To organize the items efficiently, I utilized linked lists to store and associate the jewellery items with display trays, which are further stored in display cases. This approach ensures optimal organization and retrieval of jewellery items within the system.

Techniques demonstrated:

In my Java jewellery store management system, I utilized custom-built internal data structures to efficiently manage the store. The system employs object-oriented programming principles present in some classes. Linked lists were used to organize jewellery items in display trays stored within display cases. Additionally, I implemented a user-friendly graphical interface using JavaFX components. Serialization was employed to save and load data, ensuring persistence of the jewellery store system.
